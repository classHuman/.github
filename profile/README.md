## Current Product Direction

### HADES
**Human Assisted Diagnostic Evaluation System**

A corrective safety and rehabilitation framework for autonomous AI agents.

HADES is being designed around the idea that AI systems should not simply be discarded when they fail. Instead, failures should be observable, auditable, and correctable through structured human review.

Core focus:
- AI behavior review
- agent accountability
- diagnostic workflows
- corrective feedback loops
- human final authority

---

### Aurora 24
Specialized agent infrastructure intended to support future HADES workflows and agentic operations.

Focus:
- local agent workflows
- kernel-level agent concepts
- controlled automation
- internal AI operations
